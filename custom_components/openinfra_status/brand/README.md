# Branding / Logo

Dieses Verzeichnis enthält die Branding-Dateien für die OpenInfra Status Integration.

## Dateien

| Datei | Beschreibung | Maße |
|-------|-------------|------|
| `icon.png` | Quadratisches Icon (hell) | 256x256 px |
| `dark_icon.png` | Quadratisches Icon (dunkel) | 256x256 px |
| `icon@2x.png` | Quadratisches Icon HiDPI (hell) | 512x512 px |
| `dark_icon@2x.png` | Quadratisches Icon HiDPI (dunkel) | 512x512 px |
| `logo.png` | Logo Querformat (hell) | min. 128px Höhe |
| `dark_logo.png` | Logo Querformat (dunkel) | min. 128px Höhe |

## Verwendung in Home Assistant

Ab **Home Assistant 2026.3** werden lokale Brand-Bilder aus diesem `brand/`-Verzeichnis automatisch erkannt und in der UI angezeigt — **kein externer PR nötig**.

Lokale Bilder haben Vorrang vor Bildern aus dem Brands-CDN.

Unterstützte Dateinamen:
- `icon.png` / `dark_icon.png`
- `logo.png` / `dark_logo.png`
- `icon@2x.png` / `dark_icon@2x.png`
- `logo@2x.png` / `dark_logo@2x.png`
